 📧 Contact Form Setup

The contact form uses Formspree :

1. Go to [Formspree](https://formspree.io)
2. Sign up for free account
3. Create a new form
4. Copy your form ID
5. Update the form action in `index.html`:
   ```html
   <form action="https://formspree.io/f/YOUR_FORM_ID" method="POST">
   ```
